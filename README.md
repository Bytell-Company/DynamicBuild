# About
DynamicBuild is a .NET Framework Library for Dynamic Compilation &amp; Code Generation.
Library offers multiple ways for compilation and generation.

You can compile:
- C# 
  - With CodeDom
  - With Roslyn
- VB
  - With CodeDom

You can generate:
- C# code
  - With CodeDom
  - With Roslyn
- VB code
  - With Roslyn
- IL code
  - With System.Emit
